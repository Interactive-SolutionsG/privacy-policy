# 2. Compromiso de Privacidad

# Principio Fundamental:

**Ninja Timer Control NO recopila, NO almacena, NO transmite y NO comparte ningún dato personal.**

---

# 3. Datos que NO Recopilamos

Esta aplicación **NO recopila** ninguno de los siguientes datos:

# Datos Personales:
- Nombre, apellidos, dirección
- Correo electrónico
- Número de teléfono
- Fecha de nacimiento
- Ubicación geográfica
- Dirección IP

# Datos de Uso:
- Estadísticas de uso de la aplicación
- Telemetría o análisis de comportamiento
- Horarios de uso
- Frecuencia de uso

# Datos Técnicos:
- Información del dispositivo
- Sistema operativo
- Especificaciones de hardware
- Identificadores únicos del dispositivo
- Logs de errores o diagnóstico

**Nota sobre identificadores de dispositivo:**  
Microsoft Store utiliza identificadores de dispositivo para gestionar licencias y sincronizar suscripciones entre tus dispositivos. Sin embargo, **nosotros como desarrolladores NO tenemos acceso a estos identificadores**. Microsoft Store nos proporciona únicamente una respuesta de "licencia válida" o "licencia inválida" sin revelar información del dispositivo.

# Datos de Tareas:
- Tus tareas programadas
- Horarios de ejecución
- Nombres de tareas

**TODO permanece en tu equipo local.**

---

# 4. Almacenamiento de Datos

# Datos Locales Únicamente

Toda la información que configures en Ninja Timer Control se almacena **exclusivamente en tu dispositivo local** en:

```
%APPDATA%\
```

**Archivos que se crean localmente:**
- Configuraciones de tareas
- Preferencias de usuario
- Idioma seleccionado
- Tareas programadas

**Estos archivos:**
-  Solo existen en tu computadora
-  NO se sincronizan con ningún servidor
-  NO son accesibles para el desarrollador
-  Puedes eliminarlos en cualquier momento

---

# 5. Comunicaciones con Servidores Externos

# Sin Conexión a Internet Requerida

**Ninja Timer Control NO requiere conexión a internet** para funcionar y **NO se comunica** con ningún servidor externo.

**Excepciones:**

1. **Microsoft Store:**
   - Cuando descargas o actualizas la aplicación desde Microsoft Store
   - Para validar tu suscripción o periodo de prueba
   - **Gestionado Microsoft**

2. **Actualizaciones (Opcional):**
   - Solo si eliges buscar actualizaciones manualmente
   - NO se envía información personal, solo la versión actual de la app

**NO HAY:**
- Servicios de análisis de terceros
- Servicios de publicidad
- Rastreadores
- Cookies

# 7. Permisos de la Aplicación

# Permisos Mínimos

Ninja Timer Control solicita únicamente los permisos necesarios para funcionar:

 Permiso
 **Acceso al sistema de archivos local** 
 ¿Para qué? - Guardar configuraciones y tareas
 ¿Obligatorio? - Sí 

 **Ejecución de comandos del sistema**
 ¿Para qué? - Programar y ejecutar tareas programadas
 ¿Obligatorio? - Sí 

 **Notificaciones**
 ¿Para qué? - Mostrar recordatorios de tareas
 ¿Obligatorio? - Parcialmente, es parte de la seguridad del programa hacia el usuario

 **Inicio con Windows** 
 ¿Para qué? - Funcionar en segundo plano
 ¿Obligatorio? - No, las tareas/notificaciones se ejecutan como se programan 


**NO solicitamos:**
- Acceso a cámara o micrófono
- Acceso a contactos
- Acceso a ubicación
- Acceso a fotos/videos
- Acceso a internet (excepto Microsoft licencias)

# 10. Seguridad de los Datos

# Protección Local

Dado que todos tus datos están almacenados localmente en tu dispositivo:

- **Tú eres responsable** de la seguridad física de tu computadora
- **Windows gestiona** los permisos de acceso a archivos
- **Recomendamos:**
  - Usar contraseña en tu cuenta de Windows
  - Mantener Windows actualizado
  - Usar antivirus confiable

**Por nuestra parte:**
- NO almacenamos datos en servidores
- NO hay riesgo de filtración de datos de nuestro lado
- NO hay brechas de seguridad hacia servidores

---

# 11. Derechos del Usuario

# Tus Derechos

Como NO recopilamos datos personales, no aplican la mayoría de las regulaciones de privacidad (GDPR, CCPA, etc.), pero respetamos tus derechos:

 Derecho                       ¿Cómo ejercerlo? 

 **Acceso a tus datos**        Todos están en tu dispositivo en `%APPDATA%\` 
 **Eliminar tus datos**        Desinstala la app o elimina los datos de la carpeta manualmente 
 **Portabilidad**              Copia los archivos de configuración a otro dispositivo 
 **Oposición**                 No aplica (no hay recopilación) 
 **Rectificación**             Modifica directamente los archivos de configuración, SIN modificar la estructura del programa (ingeniería inversa no permitida)
